User Guide

What This Tool Does

Reads Shell Technical Data Sheet PDFs and extracts product specifications. SKU codes are looked up automatically from EE, LV and LT product catalogues. Extracted data flows to pim.jungent.eu and from there to the Magento stores.

Shell TDS PDF → tds.jungent.eu → n8n → pim.jungent.eu → pood.jungent.eu (EE) / shop.jungent.eu (LV) / store.jungent.eu (LT)

How to Upload

  • Drag one or more Shell TDS PDFs into the upload area, or click to browse
  • Click Submit — files are processed one by one
  • Wait up to 90 seconds — result cards appear automatically
  • Review extracted data in the Results table

Result Columns

ColumnDescription
Product nameFull Shell product name
Short descriptionOne-line description from the PDF
ViscositySAE viscosity grade
ACEA / APIEuropean and American engine oil standards
SAPSLow-SAPS or Mid-SAPS (blank if standard)
FuelDiesel / Gasoline / CNG/LNG etc.
OEMOEM approvals listed in the PDF
SKU EE / LV / LTArticle codes per market

Troubleshooting

  • No results after 90 s — confirm the file is a TDS (not a Safety Data Sheet); try one file at a time
  • SKU fields empty — product name in PDF may differ from catalogue; report to administrator
  • Upload fails — file must be a PDF under 20 MB
